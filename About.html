The Program

The Indian Software ecosystem has been traditionally producing some excellent services for global corporations. However the number of entrepreneurial ventures that have come up with world renowned products are very few. In the spirit of nourishing this ecosystem, and bringing out world-class software products built from India, PayPal has parterend with "TiE Chennai" and launched this program in October 2013. As an eBay Inc. company, PayPal constantly seeks to create new opportunities around the world by focusing on building an ecosystem that nurtures entrepreneurship and innovation.
